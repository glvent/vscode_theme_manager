# Style Generator

Combine color, font, file icon, and product icon themes into named styles; fork and edit color themes; import custom icon packs.

## Testing the extension

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Compile**
   ```bash
   npm run compile
   ```

3. **Launch the Extension Development Host**
   - Press **F5** (or use **Run > Start Debugging**).
   - A new VS Code window opens with the extension loaded.

4. **Run a command**
   - In the new window, open the Command Palette (**Ctrl+Shift+P** / **Cmd+Shift+P**).
   - Run any of:
     - **Style Generator: Open Style Manager**
     - **Style Generator: Open Color Editor**
     - **Style Generator: Open Font Editor**
     - **Style Generator: Open File Icon Editor**
     - **Style Generator: Open Product Icon Editor**
   - Each command shows a short info message (full UIs are coming in later steps).

5. **Try the contributed themes** (optional)
   - **File Icons:** File Icon Theme → "Style Generator (File Icons)".
   - **Product Icons:** Product Icon Theme → "Style Generator (Product Icons)".

Close the Extension Development Host window when you’re done testing.
